This is a React/TypeScript project using Vite.

- The main source code is in the `src` directory.
- `App.tsx` is the main application component.
- `main.tsx` is the entry point of the application.
- The project uses `eslint` for linting.
- Dependencies are managed in `package.json`.
- TypeScript is configured in `tsconfig.json`.
- The public assets are in the `public` directory.
- Use tailwind css for styling
